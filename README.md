# Memory Match Game

A simple memory card matching game built with Python and Pygame.

## Features

- 5 × 6 card grid
- 15 randomized matching pairs
- Card flip mechanics
- Match detection
- Automatic card removal
- Game timer
- Win screen
- Restart button
- Quit button

## Technologies Used
- Python 3.13
- Pygame

## Installation
To Clone the repository:
git clone https://github.com/Pranish-00001/Memory-Match-Game-v0.1--Pranish


To Navigate to the project:
cd memory-match-game

To Install Pygame:
pip install pygame


To Run the game:
python main.py


## How to Play

1. Click a card to reveal its number.
2. Click a second card.
3. If the numbers match, both cards disappear.
4. If they do not match, they flip back after a short delay.
5. Match all pairs to win.
